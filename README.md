# PictureToText
图片名称写入txt里

技术：Java+JavaFx

功能：多选择图片(图二)，多拖拉图片（图三）

运行：已经打成jar包，这是路径：out/artifacts/PictureToText_jar。下载后 双击jar包即可运行。
